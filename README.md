# ani-l

Heavily inspired by [viu-media/viu](https://github.com/viu-media/viu)

Join the [Discord](https://discord.gg/zq8tgTtmWP) to discuss features, report bugs, or contribute!

A terminal-based anime library and streamer.

## Features

- Browse and search your anime library
- Stream anime directly from allanime (more sources coming soon!)
- Simple and intuitive TUI interface

## Installation

1. Ensure you have Rust and Cargo installed. If not, install them from [rustup.rs](https://rustup.rs/).
2. Run the following command to download, build, and install ani-l:
   ```bash
   cargo install ani-l
   ```
